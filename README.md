# Chinotc

## About

Chinotc is a tool to manage compilers and libraries for Chinort ([lafixier/chinort](https://github.com/lafixier/chinort)).

## Installation

You need:

- curl
- Nim

### Linux

Run the following command in your terminal:

```sh
curl "https://raw.githubusercontent.com/lafixier/chinotc/main/install.sh" -sSf | sh
```
